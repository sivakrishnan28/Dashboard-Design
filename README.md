# Task 4 - Dashboard Design

## 📌 Objective
Design an interactive dashboard for business stakeholders using Tableau Public to analyze sales performance and support business decisions.

## 🛠️ Tools Used
- Tableau Public
- Superstore Sales Dataset
- Microsoft PowerPoint

## 📊 Dashboard Features
- KPI Cards (Total Sales, Profit, Orders, Quantity)
- Sales by Category
- Monthly Sales Trend
- Top Products
- Interactive Filters (Category, Region, Order Date)

## 📈 Key Insights
- Dashboard provides a quick overview of business performance.
- Monthly sales trend helps identify seasonal patterns.
- Category-wise analysis supports product performance evaluation.
- Interactive filters enable dynamic business analysis.
